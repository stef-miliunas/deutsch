# deutsch

A German learning tracker PWA for the B1 → B2 grind. React + Vite + Tailwind, with Claude (claude-sonnet-4-6) for writing correction and speaking prompts.

## Screens

- **Today** — streak counter, manual B1→B2 progress slider, three daily habit checkboxes (Anki 15min / Listening 60min / Speaking), and a countdown to the ETH deadline (31 Mar 2027). All state lives in localStorage; checkboxes reset at midnight.
- **Write** — paste German sentences, get a minimally-corrected version with changes highlighted in green, one-sentence explanations, and a suggested Anki word.
- **Prompt** — generates a STEM/engineering-flavoured iTalki speaking prompt with 5 vocab words and 3 sentence starters.
- **Settings** (gear icon) — paste your Anthropic API key. It is stored only in localStorage and calls go directly from the browser to the Anthropic API.

## Develop

```sh
npm install
npm run dev
```

The service worker only registers in production builds, so dev stays cache-free.

## Deploy to GitHub Pages

`vite.config.js` sets `base: '/deutsch/'`, and the manifest/service worker assume that path. Build and publish `dist/` to Pages (e.g. push to a repo named `deutsch` with a Pages action, or `npx gh-pages -d dist` after `npm run build`). The app then installs to a phone home screen from the browser's "Add to Home Screen".

## Icons

`public/icons/*.png` are generated by `node scripts/gen-icons.mjs` (no image dependencies — raw PNG encoding via node:zlib).
